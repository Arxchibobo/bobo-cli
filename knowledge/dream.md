# Dream 记忆整理

定期（每天或每次长 session 结束时）做一次深度记忆整合。

## 整理流程

### 1. 扫描每日日志
读取 `memory/` 目录下最近 7 天的日志文件。

### 2. 提取模式
找重复出现的内容：
- 相同的用户偏好被记录多次 → 合并
- 相同的错误出现多次 → 晋升为硬性规则
- 相同的工作模式出现多次 → 抽象为方法论

### 3. 晋升
- 重复 2 次的纠正 → 晋升到 memory.md 业务经验
- 重复 3 次的经验 → 晋升到 knowledge 规则文件
- 有通用价值的 → 提议写入 Skill

### 4. 合并冗余
- 相似记忆条目合并为一条
- 保留最新版本，删除过期版本

### 5. 清理过期
- 已完成任务 → 从活跃任务移除
- 超过 7 天的对话摘要 → 精简或归档
- 不再适用的规则 → 标记废弃

## 触发时机
- `/dream` 命令（手动）
- 对话结束时（自动）
- memory.md 超过 5KB 时（自动瘦身）

## 归档
精简掉的内容归档到 `memory/archive/` 目录，不直接删除。
