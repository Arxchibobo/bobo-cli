# csharp-lsp

C# language server for Claude Code, providing code intelligence and diagnostics.

## Supported Extensions

`.cs`

## Installation

### Via .NET tool (recommended)

```bash
dotnet tool install --global csharp-ls
```

### Via Homebrew (macOS)

```bash
brew install csharp-ls
```

## Requirements

- .NET SDK 6.0 or later
